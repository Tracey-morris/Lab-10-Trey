# Lab-10-Trey

## Overview
This is a client-side application for INFOST 270-201 Lab assignment. The application is built with vanilla HTML, CSS, and JavaScript and is designed to run entirely in the browser without any server-side dependencies.

## Features
- **Pure Client-Side**: No server required - runs entirely in the browser
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Interactive Elements**: Button with click tracking and dynamic output
- **GitHub Pages Ready**: Can be hosted directly on GitHub Pages

## Files

### index.html
The main HTML file that provides the structure and content for the application. It includes:
- Header with title and description
- Content section with application information
- Interactive section with a clickable button
- Footer with copyright information
- Links to CSS and JavaScript files

### styles.css
Contains all styling for the application:
- Gradient backgrounds and modern design
- Responsive layout using flexbox principles
- Button hover and active states with animations
- Mobile-friendly media queries
- Smooth transitions and animations

### script.js
Implements the interactive functionality:
- Button click event handler
- Click counter that tracks the number of clicks
- Dynamic output messages based on click count
- Console logging for debugging

## How to Run

1. Clone the repository
2. Open `index.html` in your web browser
3. Or visit the GitHub Pages URL once enabled in repository settings

## How to Enable GitHub Pages

1. Go to repository Settings
2. Scroll to "GitHub Pages" section
3. Select "main" branch as source
4. Save

Your application will be available at: `https://Tracey-morris.github.io/Lab-10-Trey/`
